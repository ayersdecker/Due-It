# Due It
> Description

This personal organization app is designed for students who manage between classes, homework, and other life obligations. Due It offers a lightweight solution to manage your lifestyle and has smart features to help optimize when you work on assignments. 

> Whats New?

Due It currently features Smart Scheduling to automatically fill timeslots for assignment/project work sessions that don't interfere with other events and helps you complete things early! Due It will help push you to complete things sooner rather than later, because a relaxing weekend is always nice. 

> Current Version: 

Due It Version 1.0 is currently under development. It will be released to Android systems that support Q (Android 10.0) and newer.
